## Development Installation


## Watch for Changes

```bash
npx tailwindcss -i ./css/style.css -o ./dist/output.css --watch
```

## Build CSS

```bash
npx tailwindcss -i ./css/style.css -o ./dist/output.css # --minify
```
